# Support

## Where to get help

- **Bugs**: open an issue on GitHub
- **Feature requests**: open an issue with the feature-request template
- **Questions**: GitHub Discussions
- **Security**: see SECURITY.md for private reporting

## Response expectations

This is a community project maintained on a best-effort basis. Response times vary. Issues are triaged in order of severity and impact.
